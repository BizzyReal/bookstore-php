# 📚 Bookstore PHP Native USK

Project website toko buku menggunakan PHP Native dan MySQL.

## 🛠 Teknologi yang Digunakan

- PHP Native
- MySQL
- Bootstrap
- HTML
- CSS
- XAMPP

## 📂 Cara Menjalankan Project

1. Download project dari GitHub
2. Simpan folder `bookstore` ke dalam:

C:\xampp\htdocs

3. Jalankan XAMPP:
   - Start Apache
   - Start MySQL

4. Import database:

- Buka http://localhost/phpmyadmin
- Buat database dengan nama:

bookstore

- Import file:

bookstore.sql

5. Jalankan di browser:

http://localhost/bookstore
